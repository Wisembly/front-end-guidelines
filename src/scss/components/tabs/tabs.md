# Tabs

```html
<ul class="Tabs">
  <li class="Tab-item" aria-selected="true">
    <a class="Tab-link" tabindex="0" href="#foo">
      Foo selected
    </a>
  </li>
  <li class="Tab-item" aria-selected="false">
    <a class="Tab-link" tabindex="-1" href="#bar">
      Bar thing thing
    </a>
  </li>
  <li class="Tab-item" aria-selected="false">
    <a class="Tab-link" tabindex="-1" href="#baz">
      Baz thing thing
    </a>
  </li>
</ul>
```
