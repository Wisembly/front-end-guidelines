# Tabs

```html
<ul class="Tabs">
  <li class="Tab-item is-selected" aria-selected="true">
    <a class="Tab-link" tabindex="0" href="#foo">
      Foo selected
    </a>
  </li>
  <li class="Tab-item">
    <a class="Tab-link" aria-selected="false" tabindex="-1" href="#bar"  aria-selected="false">
      Bar thing thing
    </a>
  </li>
  <li class="Tab-item">
    <a class="Tab-link" aria-selected="false" tabindex="-1" href="#baz"  aria-selected="false">
      Baz thing thing
    </a>
  </li>
</ul>
```
