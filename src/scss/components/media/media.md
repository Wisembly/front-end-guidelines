# Media

## Basic markup

```html
<figure class="Media">
	<img class="Media-img Avatar Avatar--s" src="https://upload.wikimedia.org/wikipedia/en/6/69/Ember.js_Logo_and_Mascot.png" />
	<figcaption class="Media-content">Aenean lacinia bibendum nulla sed consectetur.</figcaption>
</figure>
```
