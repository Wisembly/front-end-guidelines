# Dropdown

## Basic

```html
<div style="position: relative; border: 1px dashed #aaa; height: 50px;">
	<ul class="Dropdown">
	  <li class="Dropdown-item">
	    Simple label
	  </li>
	  <li>
	    <a href="#" class="Dropdown-item">
	      Linked label
	    </a>
	  </li>
	</ul>
</div>
```
